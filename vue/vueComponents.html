    Vue 组件使用

    # Vue 组件使用

    # 1. props 和 $emit

    1. props: 父组件想子组件传递信息
    2. $emit: 子组件触发父组件传递的事件

    # 2. 组件间通讯 - 自定义事件

    1. 父子组件通讯：参考知识点1. props 和 $emit

    2. 兄弟组件通讯：自定义事件

      event.$emit("doSomething", params) // 触发

      event.$on("doSomething", this.doSomething) //接收

      // 解绑。及时销毁，避免造成内存泄漏

      event.$off("doSomething", this.doSomething)

    # 3. 组件生命周期

    1. 单个组件

      挂载阶段

      1. beforeCreate
      2. created // vs mounted vue实例初始化完成，但页面还没有开始渲染
      3. beforeMount
      4. mounted // vs created 页面已经渲染完成，可以操作 dom

      更新阶段

      1. beforeUpdate
      2. updated

      销毁阶段

      1. beforeDestroy // 解除绑定，销毁子组件以及事件监听器
      2. destroyed

    2. 父子组件

      初始化实例（created）：父 --> 子

      页面渲染（mounted）：子 --> 父

      父beforeCreate => 父created => 父beforeMount => 子beforeCreate => 子created => 子beforeMount => 子mounted => 父mounted

      更新实例（beforeUpdate）：父 --> 子

      更新完成（updated）：子 --> 父

      父beforeUpdate => 子beforeUpdate => 子updated => 父updated

      销毁子组件（beforeDestroy）：父beforeUpdate => 子beforeDestroy => 子destroyed => 父updated
